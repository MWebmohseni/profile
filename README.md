# رزومه لیلا محسنی - توسعه‌دهنده وب (WordPress & Node.js)

توسعه‌دهنده وب با تمرکز بر وردپرس و جاوااسکریپت، مسلط به طراحی قالب و افزونه‌های اختصاصی. دارای تجربه در پیاده‌سازی سیستم‌های مالی و حسابداری.

## 🛠 مهارت‌های فنی

- **توسعه وب:** JavaScript (ES6+), HTML5, CSS3 (Responsive Design)
- **مدیریت محتوا (CMS):** تسلط کامل بر وردپرس (توسعه قالب و شخصی‌سازی افزونه‌ها)
- **تکنولوژی‌های بک‌اند:** Node.js, Express.js
- **پایگاه داده:** MySQL, MariaDB
- **ابزارها:** Git, XAMPP/Apache

## 🚀 پروژه‌های منتخب

### سیستم حسابداری اختصاصی طلا
- توسعه یک سیستم حسابداری کاملاً شخصی‌سازی شده برای مدیریت تراکنش‌های طلا.
- استفاده از **Node.js** برای مدیریت دقیق داده‌های مالی.
